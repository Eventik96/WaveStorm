RÖVID LEÍRÁS:
Ez egy online zene stream weboldal. A spotify-hoz hasonlítható leginkább.
Az adatbázisban található zenéket le tudjuk játszani, véletlenszerûen tud ajánlani nekünk zenéket és a kedvenc
zenéinket lejátszási listákba rendezhetjük.

//////////////////////////////////////////////////
HASZNÁLT TECHNOLÓGIÁK:
- PHP
- HTML5
- CSS
- Javascript(jQuery-vel)
- MySQL
- AJAX

///////////////////////////////////////////////////
MEGJEGYZÉSEK:
A Facebookos regisztráció, illetve bejelentkezés nem fog mûködni más gépen futtatott webszerveren, mert
a hozzá tartozó Facebook APP-ot amit csináltam a https://developers.facebook.com oldalon, az én IP-címemhez rendeltem.
Egy alkalmazáshoz meg csak egy host címet lehetséges hozzárendelni.
